# Selenium Automation Framework (Python + PyTest + POM + CI/CD)

This project is a complete **UI automation testing framework** built using **Python, Selenium, PyTest, and Page Object Model (POM)**.  
It also includes **Data Driven Testing**, **Config Management**, **Screenshot on failure**, and **CI/CD integration using GitHub Actions**.

The framework automates user scenarios on:
https://automationexercise.com

---

## Features

- Page Object Model (POM) design pattern
- PyTest framework for test execution
- Data Driven Testing using Excel
- Config file handling using JSON
- Screenshot capture on test failure
- End-to-End (E2E) test coverage
- CI/CD integration using GitHub Actions
- Headless browser execution in CI
- Clean and scalable folder structure

---

## Tech Stack

- Python
- Selenium WebDriver
- PyTest
- OpenPyXL (Excel handling)
- GitHub Actions (CI/CD)
- JSON (Config management)
